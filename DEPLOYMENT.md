## 🔧 Customization

### Update WhatsApp Number
Edit `app/page.tsx`:
```tsx
href="https://wa.me/6281234567890?text=..."
```

### Change Pricing
Edit pricing section in `app/page.tsx`:
```tsx
// Starter: Rp 500K
// Business: Rp 1.5Juta
// Premium: Rp 3Juta
```

### Add Custom Domain
1. Go to Vercel Dashboard → Project Settings → Domains
2. Add domain (e.g., `webumkm.id`)
3. Update DNS records
